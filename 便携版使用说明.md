# TexCat 便携版使用说明

这份说明面向直接下载 `TexCat_版本号_portable.zip` 的用户。便携版已经包含运行环境，解压后可以直接使用，不需要安装 Python。

## 快速启动

1. 解压 `TexCat_版本号_portable.zip` 到任意文件夹。
2. 双击 `TexCat.exe`。
3. 浏览器会自动打开 `TexCat贴图工具箱` 页面。
4. 使用完成后，点击左上角电源图标 `退出工具箱`，或直接关闭工具箱浏览器窗口等待后台服务自动退出。

建议不要把工具箱解压到系统保护目录，例如 `C:\Program Files`。如果需要长期使用，可以放在桌面、项目工具目录或独立软件目录。

## 导入图片

- 可以把图片直接拖入页面顶部的虚线区域。
- 可以点击 `批量选择图片` 手动选择单张或多张图片。
- 可以选择 `自定义输入目录` 批量读取某个文件夹内的图片。
- 拖入或批量选择图片时，工具箱会使用临时缓存，不会修改原始图片。

支持输入格式：

```text
PSD / PNG / TGA / TIF / TIFF / JPG / JPEG / DDS / WEBP / BMP
```

PSD 输入会读取当前合成图像，不会读取或保留原始图层结构。

## 输出位置

界面里的输出位置可以选择：

- `默认输出文件夹`：输出到工具箱目录下的 `output` 文件夹。
- `源文件位置`：输出到每张源图所在文件夹，仅适用于自定义输入目录。
- `导出时弹框选择`：执行时再选择输出目录。
- `自定义输出目录`：固定输出到指定目录。

如果目标位置已经存在同名文件，工具箱会弹窗确认，可以选择覆盖、取消，或给本次输出整体添加 `_TC` 后缀。

## 常用模块

- `图像大小`：按 1K、2K、4K、8K、512、256 或自定义尺寸批量缩放图片。
- `格式转换`：在 PSD、PNG、TGA、TIF、JPG、DDS、WEBP、BMP 等格式之间互转。
- `高质量压缩`：在尽量保持视觉质量的前提下重新写出压缩版本。
- `图片裁切`：支持单图裁切，也支持多张同尺寸图片按同一位置批量裁切。
- `法线/黑白图调整`：调整法线强度、OpenGL/DX 方向，以及黑白图强度、色阶、Gamma、曲线等。
- `PBR辅助转换`：从颜色图、法线图或高度图生成 Normal、Height、AO、Cavity、Curvature 等辅助贴图。
- `通道拆分`：把图片拆成 R/G/B/A 或灰度通道，并支持自定义通道输出命名。
- `通道合并`：把多张单通道图或指定通道重新打包成 RGB/RGBA 贴图。
- `批量重命名`：使用步骤叠加方式批量查找替换、添加前缀、添加后缀或中间插入文本。

## 通道和质量提醒

- 工具箱统一按 `8位/通道` 写出图片。
- `自动/跟随源图` 会尽量保持源图常见模式：灰度保持灰度，RGB 保持 RGB，带透明通道的图保持 RGBA。
- `JPG/JPEG` 本身是有损格式，不支持 Alpha 透明通道。
- `PNG / TGA / TIF / TIFF / BMP / PSD` 更适合需要保留无损或贴图生产属性的流程。
- `DDS` 当前使用运行时默认写出能力，不等同于专门的 BC1/BC3/BC7 游戏贴图压缩器。

## 日志和备忘录

- 左侧 `日志` 会显示当前操作、导出结果和格式提醒。
- `清空日志` 只清空页面日志，不会删除文件。
- `备忘录` 保存在当前浏览器本地，可以临时记录项目命名、通道规则或导出要求。

## 常见问题

如果双击 `TexCat.exe` 后没有打开浏览器，可以手动访问：

```text
http://127.0.0.1:8765/
```

如果页面提示端口被占用，可以先关闭其他 TexCat 窗口，或点击旧页面左上角 `退出工具箱`。

如果导出失败，先查看左侧日志。常见原因包括：目标文件被其他软件占用、输出目录没有写入权限、源图格式损坏，或输入图片尺寸过大导致内存不足。
