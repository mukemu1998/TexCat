# TexCat 便携版

TexCat 便携版开箱即用。解压后直接双击 `TexCat.exe`，不需要安装 Python，不需要运行 pip，也不需要配置环境。

## 快速使用

1. 解压 `TexCat_版本号_portable.zip`。
2. 双击 `TexCat.exe`。
3. 把图片拖入页面顶部虚线区域，或点击 `批量选择图片`。
4. 选择功能模块、输出格式和输出位置。
5. 点击 `执行并导出所有选择的贴图`。
6. 到 `output` 或你选择的目录查看结果。

拖入或选择图片不会修改原图；默认输出目录是工具箱文件夹里的 `output`。

## 关闭工具箱

点击页面左上角电源图标 `退出工具箱`，会立即关闭当前工具箱服务。

也可以直接关闭浏览器窗口，后台服务会在页面断开后自动退出。

## 输出位置

- `默认输出文件夹`：输出到工具箱目录下的 `output`。
- `源文件位置`：输出到每张源图所在文件夹，仅适用于自定义输入目录。
- `导出时弹框选择`：执行时再选择输出目录。
- `自定义输出目录`：固定输出到指定目录。

如果目标位置已有同名文件，工具箱会弹窗确认，可选择 `覆盖`、`取消`，或整体添加 `_TC` 后缀。

## 支持格式

```text
PSD / PNG / TGA / TIF / TIFF / JPG / JPEG / DDS / WEBP / BMP
```

PSD 输入读取当前合成图像；输出 PSD 为扁平图像，不保留原始图层结构。

## 功能模块

- `图像大小`：批量缩放图片。
- `格式转换`：常用贴图格式互转。
- `高质量压缩`：尽量保持视觉质量的压缩输出。
- `图片裁切`：单图裁切或多图同位置批量裁切。
- `法线/黑白图调整`：调整法线强度、黑白图强度、色阶、Gamma 和曲线。
- `PBR辅助转换`：生成 Normal、Height、AO、Cavity、Curvature 等辅助贴图。
- `通道拆分`：拆出 R/G/B/A 或灰度通道。
- `通道合并`：将单通道或指定通道重新打包成 RGB/RGBA。
- `批量重命名`：按步骤叠加批量改名。

## 注意事项

- 工具箱统一按 `8位/通道` 写出图片。
- `JPG/JPEG` 是有损格式，不支持 Alpha 透明通道。
- `PNG / TGA / TIF / TIFF / BMP / PSD` 更适合需要保留无损或贴图生产属性的流程。
- 图片处理功能本地运行，没网也能用；天气、节日、位置等在线信息可能不显示。

## 常见问题

如果双击后没有自动打开浏览器，可以手动访问：

```text
http://127.0.0.1:8765/
```

如果页面提示端口被占用，先关闭其他 TexCat 窗口，或点击旧页面左上角 `退出工具箱`。

如果导出失败，先看左侧日志。常见原因包括文件被占用、输出目录没有写入权限、源图损坏或图片过大导致内存不足。
