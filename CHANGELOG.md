# CHANGELOG

## v1.02-dev

开发中。

- 整理开源前文档和启动脚本，移除个人本地路径，补充通用安装说明。
- 新增 `LICENSE` 和 `requirements.txt`，仓库按 MIT License 开源准备。
- 保持 `main` 分支上的 `v1.01` / `v1.01.1` 作为稳定回滚点，在 `dev/v1.02` 分支继续整理。
- 新增 `WORKFLOW_DESIGN.md`，先固定工作流模式 Beta 的设计边界、步骤模型和迭代顺序。
- 新增 `工作流模式 Beta` 入口和三栏空壳页面，只同步资源池信息，不接真实处理逻辑。
- 工作流模式 Beta 支持添加、选择、删除、复制、上移、下移步骤，并支持保存/载入工作流 JSON。
- 工作流模式 Beta 开始接入低风险步骤参数：缩放尺寸、格式与压缩、命名规则可编辑参数、读取当前快捷模块设置并写入工作流 JSON。
- 工作流模式 Beta 新增输出预览：可预估缩放/格式/命名后的输出文件名、目标路径和同名冲突，不写出图片。
- 图片裁切编辑模式新增适配、缩放、1:1 和移动视图控制，长图进入裁切模式后可拖动查看并继续框选。
- 当前开发轮次先把 `PBR辅助转换` 排除在工作流推进范围外；PBR 继续保留在快速工具模式，等后续单独优化后再重新接入工作流。
- 工作流模式 Beta 新增首条真实执行链：`缩放 -> 格式与压缩 -> 命名规则 -> 导出`，并复用全局输出位置、同名冲突检查和 `_TC` 后缀策略。

## v1.01.1

稳定版补丁。

- README 增加项目 Logo 展示，并同步清理已移除的无黑窗 VBS 启动说明。
- 移除 `无黑窗启动TexCat.vbs`，保留 `启动TexCat.bat` 作为 Windows 启动入口。
- 增加 Python 3.14 表单解析兼容，避免新版 Python 缺少 `cgi` 模块导致工具箱启动失败。
- 保持 `v1.01` 的功能范围不变，只修正开源发布和启动兼容相关内容。

## v1.01

当前稳定基线版本。

- 保留快速工具模式：图像大小、格式转换、高质量压缩、图片裁切、法线/黑白图调整、PBR辅助转换、通道拆分、通道合并、批量重命名。
- 支持 PSD / PNG / TGA / TIF / TIFF / JPG / JPEG / DDS / WEBP / BMP 输入输出。
- 支持默认输出目录、源文件位置、导出时选择、自定义输出目录。
- 支持导出前同名冲突检查、覆盖/取消/_TC 后缀处理。
- 支持通道/位深策略：自动、RGB 24位、RGBA 32位、灰度 8位。
- 支持裁切编辑、1:1 锁定、自定义像素裁切、九宫格吸附。
- 支持左侧备忘录、日志、主题设置、退出工具箱、默认输出目录清空。
